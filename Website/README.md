# DataProtector Website

This directory contains the standalone bilingual project website for
DataProtector. It is a static open-source homepage and is intentionally separate
from `DataProtectorWebAdmin`, which is the product operator console.

本目录是 DataProtector 的独立中英双语官网，用于介绍项目业务价值、核心能力、
工作闭环、平台组成和开源定位。它不是 `DataProtectorWebAdmin` 管理后台。

## Preview

Open `index.html` directly in a browser. No build step is required.

```text
index.html?lang=zh
index.html?lang=en
```

The language switch also stores the selected language in `localStorage`.
